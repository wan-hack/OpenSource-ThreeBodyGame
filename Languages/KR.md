# 오픈소스 『삼체』 게임 — 1인칭 체험판

## 🌟 프로젝트 소개
이것은 류츠신의 『삼체』를 사랑하는 전세계 팬들을 위해 제작된**비영리·무료 오픈소스 게임**입니다. 소설에서 왕묘가 경험한 1인칭 게임 체험을 완벽하게 재현하여, 삼체성에 몰입하고 항기원·란기원 사이에서 탈수·침수를 핵심 메커니즘으로 생존을 건립니다.

## 🎮 핵심 게임플레이: 1인칭 몰입형 생존
1.  **시점**：1인칭 시점 (당신이 게임 내 플레이어), 삼체성의 간단한 마을에 위치합니다.
2.  **기원 규칙**：항기원/란기원을 무작위로 시작. 란기원 5초 전에 빨간 텍스트 프롬프트(「란기원 도래! 극한의 더위, 신속히 탈수하라!」)가 표시됩니다.
3.  **탈수·침수**：
    - 항기원：자신을 클릭하여 침수 (활동 재개)하고, 수원/광석 자원을 수집.
    - 란기원：자신을 클릭하여 탈수 (건조 섬유가 되어, 극한의 더위에 면역)하고, 항기원 복구 후 다시 침수.
4.  **승리 조건**：혼합 자원 3개를 수집하고, 마을의 석조 천문대에 가서 기원 예측 기능을 언락하여 «문명의 생존»을 달성.
